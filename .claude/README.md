# .claude

Personal project profile for **Claude Code**, alongside the other agent
profiles in this repo (`.devin/`, `.opencode/`, `.windsurf/`).

This entire folder is **gitignored** (see the `# Claude Code` entry in the
root `.gitignore`) — it holds local-only agent configuration and scratch work
that should not be committed.

## Layout

- `Plans/` — working plan documents for in-progress tasks
- `workflows/` — reusable workflow / slash-command definitions
- `prompts/` — saved agent prompt templates
- `settings.local.json` — local Claude Code settings (permissions, etc.)
